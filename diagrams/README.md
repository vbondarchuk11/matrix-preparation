# Project Diagrams

This folder contains evaluation-ready diagrams for the Matrix CRM frontend project.

Recommended set for presentation:

1. `c4-system-container.mmd`
   C4-style system and container overview

2. `use-case-diagram.mmd`
   UML-style use case diagram for key user actions

3. `component-diagram.mmd`
   UML-style component diagram for the frontend architecture

4. `entity-relationship-diagram.mmd`
   ER diagram for the main CRM domain entities

Notes:

- Files are written in Mermaid syntax for easy editing.
- Some UML notations are represented in Mermaid-compatible form.
- Existing diagrams in this folder are still available as additional material.
